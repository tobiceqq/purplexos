# Modular Sci-Fi Bio Lab Pack
# by Hani Jahan Design

A clean, stylized modular tile set designed for worldbuilding and rapid prototyping of 3D platformers, puzzle games, and grid-based levels in Unity. Optimized for clarity, modularity, and performance, using a single shared color palette across all assets.

##  How to Use

1. Import into any Unity 2020.3+ project.
2. Drag prefabs into your scene from:
   `Assets/HaniJahanDesign/SciFiBioPack/Prefabs/`
3. Assign the correct material:
   - **URP**  `HJD_SFB_URP.mat`
   - **Built-in RP**  `HJD_SFB_Builtin.mat`
4. All tiles are pivot-centered and grid-aligned for easy snapping.

---

##  Tech Notes

- All assets are FBX
- Tri-count: most are around 100-300 tris
- Materials use only Base Color maps
- No normal, roughness, metallic, or emission maps
- No custom shaders or scripts
- Works with both URP and Built-in RP

---

##  Compatibility

- Unity 2020.3 LTS or newer
- Supports: Built-in RP and URP
- No HDRP, no external dependencies
